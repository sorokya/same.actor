# Same Actor

Is that the same actor from another movie you remember? Use Same.Actor to find out!

# How to run

Install dependencies:

`npm install`

Install api dependencies:

`cd api && npm install`

Provide The Movie Database API Key in `api/.env`

`TMDB_KEY=<YOUR_API_KEY>`

Start the api:

`npm run dev-api`

Start the app:

`npm run dev`
